# 株価上昇予測システム

LightGBMによる日本株の短期上昇予測システム。

## 成績

- 勝率: 66.7%
- 総リターン: +57.70%
- PF: 2.13

## 売買ルール

- エントリー: スコア上位1銘柄を翌日寄付きで買い
- 利確: +10%
- 損切: -10%
- 最大保有: 20営業日

## 実行方法

```bash
# 学習
.venv/bin/python scripts/phase2_train.py

# バックテスト
.venv/bin/python scripts/phase3_backtest.py
```
