# 株価上昇予測システム

LightGBMモデルによる日本株の短期上昇予測システムです。

## 機能

- 日付を選択して特定日の買い候補を表示
- 買い理由の自動分析（RSI、出来高、続落など）
- スマートフォン対応

## 売買ルール

- 利確: +12%
- 損切り: ATR×2.0
- 最大保有: 15日

## 注意事項

このシステムは過去データに基づく機械学習モデルの予測であり、将来の株価を保証するものではありません。投資判断は自己責任でお願いします。
